Sec. 3 Intro to HTML notes

HTML Attributes

Attributes so far...

  • src
  • alt (screen readers can read the text)

Lists

Use ol tag when the order matters like in a recipe.

Links

  • Allow users to click from page to page and access resources around the web
  • Links can be text, images or other elements.
  • href attribute specifies destination of a link.

Comments

  • Used to explain your code.
  • You can comment out code.
  • You could come back to your code months in the future and read you comments to understand your code.
